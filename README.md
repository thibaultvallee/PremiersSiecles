# Christianisme (Iᵉʳ–Vᵉ siècle)

Outil pédagogique interactif pour l’étude des cinq premiers siècles du christianisme.

## Contenu
- Frise chronologique interactive (auteurs et événements)
- Table des auteurs (triable, filtrable)
- Table des événements (triable, filtrable)
- Fiches informatives accessibles par clic (modal)

## Périmètre
- Ier au Ve siècle
- Auteurs chrétiens majeurs
- Conciles, hérésies, édits impériaux

## Conventions
- Dates approximatives indiquées par "ca."
- Langue : grec / latin
- Villes = lieux d’activité attestés ou traditionnels

## Structure des données
- `data/auteurs.json`
- `data/evenements.json`
- `data/relations.json`

## Publication
Site statique, publiable sur GitHub Pages.
